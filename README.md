## Aim:
To write a python program to find the inverse of a matrix
## Equipment’s required:

## Algorithm:
### Step1 : Import the numpy module to use the built-in function for calculation
### Step 2: Prepare the list from matrix abd assign in np.array()
### Step 3:Using the np.linalg.inv(),we can calculate the solution. 
### Step 4: End the program

## Program:
#Program to find the inverse of a matrix.
#Developed by: A.Aira Dario
#RegisterNumber:25016155
```
import numpy as np
A=np.array([[2,1,1],[1,1,1],[1,-1,2]])
X=np.linalg.inv(A)
print(X)
```
